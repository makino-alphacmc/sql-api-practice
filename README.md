# sql-api-practice

SQLとFastAPI(CRUD API)の練習用リポジトリ。データベースは`SQL/`と`API/`で共有している。SQL練習とAPI練習は別々のセッションとして進める。

- [`SQL/`](SQL/) — PostgreSQL(`ec_practice`)を使ったSQL練習。スキーマ・データ・出題ガイドは`SQL/README.md`と`SQL/EXAM.md`を参照。「**SQLスタート**」と言うと開始する。
- [`API/`](API/) — FastAPIでのCRUD API実装練習（controller / service / model構成）。出題ガイドは`API/API_EXAM.md`を参照。「**APIスタート**」と言うと開始する。

どちらも、まず接続/起動方法を案内 → 確認 → 問題出題、という流れで進める。
