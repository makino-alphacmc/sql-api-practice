# API練習 出題履歴

このファイルは「APIスタート」で出題した問題を1問ずつ記録するログです。
新しいセッションで「APIスタート」と言われたら、出題AIはまずこのファイルの末尾（直近の問題）を確認し、
- 直近のリソース・CRUD操作・パラメータパターン
- 直近の難易度

を踏まえた上で、少しずつ難易度・パターンを広げながら次の問題を出すこと。

記録ルール:
- 問題を出題した時点で1エントリ追加する（結果は「未回答」にしておく）
- ユーザーが実装・curl確認の結果を報告したら、そのエントリの「結果」を更新する（✅正解 / ⚠️惜しい / ❌不正解）
